# Verificador de Número Par ou Ímpar em Java

## 📌 Sobre o projeto
Este é um projeto simples desenvolvido em **Java** para praticar **lógica de programação**.

O programa verifica se um número é **par ou ímpar** utilizando:
- Estrutura condicional `if/else`
- Operador módulo `%`

## ⚙️ Como funciona

O programa define um número inteiro e verifica o resto da divisão por 2.

- Se o resto for **0** → o número é **par**
- Caso contrário → o número é **ímpar**
